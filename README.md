# 医院排队叫号分诊管理系统

#### 介绍
医院排队叫号分诊管理系统，包含6个端：管理端、分诊端、医生端、大屏端、签到端（自助机）、诊室屏

#### 软件架构
使用C#开发，数据库为SQLServer
